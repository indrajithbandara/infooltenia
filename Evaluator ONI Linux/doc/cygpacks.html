While some of the following packages might not be absolutely required, they are likely to prove useful. I  recommend that you install:

  • ash
  • autoconf
  • automake
  • bash
  • bc
  • binutils
  • bzip2
  • clear
  • cygutils
  • cygwin
  • cygwin-doc
  • diff
  • fileutils
  • findutils
  • gcc
  • gdb
  • gdbm
  • gettext
  • grep
  • groff
  • gzip
  • inetutils
  • less
  • make
  • man
  • mingw-runtime
  • ncurses
  • openssl
  • patch
  • pcre
  • readline
  • sed
  • sh-utils
  • tar
  • termcap
  • terminfo
  • textutils
  • w32api
  • which
  • zlib

I also recommend that you install joe, a friendly text editor with a Word Perfect-like interface (Borland IDEs also use this interface). It can be downloaded by following a link from the Cygwin home page.
